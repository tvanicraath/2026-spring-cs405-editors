## How to Apply
1. Only around 8 editors will be selected, so hurry up!
2. Fork this repo.
3. Create a file: `editors/<YOUR_ROLL_NUMBER>.yaml` (Roll number must be in all CAPS.)
4. Copy the content below and fill in your details:

```yaml
# editors/UI24CS00.yaml

name: "Rachit Nimavat"  #Your name as you want it to appear
roll: "UI24CS00"    #In all CAPS
github: "tvanicraath"      #Your GitHub username that will be used to fetch profile pic
email: "rachit@420840.xyz"  #Your permanent or institute email
website: "https://nimav.at" #Link to your website/LinkedIN/portfolio/blog/Instagram/Snapchat/etc.
```

5. Create a Pull Request.
6. Once your PR is merged, you will be added to the editorial board.
